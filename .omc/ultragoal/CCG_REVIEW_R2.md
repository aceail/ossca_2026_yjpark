## D. Gemini 단독 — Tier 2 (임상·문화·연출)

1. **2인칭 → 1인칭 미래 자아 강제** — "너는 ~할 것이다" 금지 → "**나는 지금 ~하고 있다**" 미래 자아 1인칭 화법
2. **절대 금지어 리스트** — 의지·노력·한심·정상·게으름 (한국 가스라이팅 어휘)
3. **Self-Destruct 버튼** — 세션 단위 즉시 삭제, 입력 솔직함 회복
4. **사회적 노출 보호** — 가족·연인이 화면 봤을 때 페르소나 붕괴 대응. 화면 잠금 시 카드 자동 블러
5. **역설적 의도 감지** — 사용자가 앱 비꼴 때 정서적 항복 ("지금 많이 힘드시군요") 모드 전환
6. **특수 상태 검증 페르소나 5개** — 새벽 2시 자기파괴·신체 통증·경제 위기·알코올/약물·Productive Procrastination
7. **한국 정서 3사각** — 억울함 기반 미루기·체면+완벽주의·정서적 허기/공허
