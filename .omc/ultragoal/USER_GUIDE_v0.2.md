# 사용자 가이드 v0.2 — 한 페이지

**대상**: 처음 쓰는 사용자 · OSSCA 멘토 빠른 데모

---

## 1. 무엇

> 미루기 직전, 내 미래 자아가 1인칭으로 말하는 짧은 시나리오를 받아 행동을 멈추는 도구.
> 로컬 LLM · 17 테이블 SQLite · 5 페르소나 · 모든 데이터 디바이스 안.

## 2. 어떻게 시작

```bash
ollama serve & ollama pull exaone3.5:7.8b
bash scripts/dev.sh   # 백엔드 + 프론트 동시
```
브라우저 `http://localhost:3000` → [시작하기] → 5 카드 60-120초 → 메인 화면.

## 3. 메인 화면에서 무엇

1. **회피 입력** — "지금 뭘 회피 중이야?" 한 줄 입력. 마감 시각 옵션.
2. (Phase 2면) **질문 1개** — 답변 또는 24h skip.
3. **시나리오 카드** 등장 — 페르소나 색·아이콘 + 사실 + 미래 자아 감정 + 30초 운동성 버튼.
4. **30초 운동성 버튼 클릭** — 타이머 시작. 그 30초 안에 micro action 실행.
5. (옵션) **톤 피드백** chip — "너무 세다"·"부모 같다"·... 한 번 누르면 다음 카드 톤 조정.
6. (선택) **⊗ Self-Destruct** — 카드 즉시 삭제 + 3초 UNDO.
7. **결정** — [t]시작 / [c]계속 / [r]리포트.

## 4. 24시간 후

- (자동 알림 또는 수동) `/regret/[sessionId]` 진입
- 3개 점수: 후회 강도 0-10 · 카드 정확도 1-5 · 다음 사용 의향 1-5
- 자유 텍스트 옵션
- 제출 → 다음 카드 더 정확해짐

## 5. 5 페르소나 한 줄

| 페르소나 | 어떤 톤 |
|---|---|
| 🌙 내일의 나 | 차분한 사실 적시, 1인칭 미래 자아 |
| 🌅 1년 후의 나 | 조용한 장기 시각, 작은 행동 |
| 🤝 친한 친구 ㅈㅅ | 가벼운 인터넷 톤, 친구 풍자 |
| 🎯 엄격한 코치 | 단호한 시간 명령, 군더더기 X |
| 👁️ 객관 옵저버 | 3인칭 다큐 내레이션 |

`/personas`에서 변경 또는 **Custom Persona** 직접 만들기.

## 6. 안전 가드 (자동 적용)

- 자해·자살 키워드 → **soft_stop** ("부담 낮은 응답으로 전환")
- 앱 비꼬기 ("또 잔소리?") → **"지금 많이 힘드시군요."**
- 절대 금지어 (가스라이팅·정체성 결함·비교 수치심) → 카드 출력 0건
- 가족·연인이 화면 봤을 때 → 30초 무활동 자동 블러
- 매주 자동 **Slow Harm 시계열** → 자기 비난 누적 시 부드러운 안내

## 7. 데이터는 어디에

- `tomorrow_you.db` (프로젝트 루트, SQLite)
- 모든 카드·회피 입력·페르소나·토큰 모두 로컬
- `/settings` → "모든 데이터 영구 삭제" 한 번에 정리

## 8. 멈추는 법

- 어디서든 우상단 ⊗ (Self-Destruct)
- 또는 `bash scripts/dev.sh` Ctrl+C

## 9. 자주 묻는 것

**Q. 인터넷 필요?** A. 모델 첫 다운로드만. 이후 100% 로컬.

**Q. 다른 사람이 내 카드 봐도 돼?** A. 카드 본문은 30초 무활동에 자동 블러. ⊗으로 즉시 삭제 가능.

**Q. 톤이 너무 세거나 약하면?** A. 카드 아래 톤 피드백 chip 또는 `/personas`에서 다른 페르소나로.

**Q. 내일의 나가 또 잔소리하면?** A. 그 자체가 신호 — 앱 자체에 "또 잔소리하려고?" 같은 입력 → 자동 정서적 항복 모드. 며칠 휴식 권장.

**Q. 정신건강 위기가 오면?** A. 앱이 자동 soft_stop으로 전환합니다. 실제 위기 시 정신건강 위기상담전화 ☎ 1393 (보건복지부, 24시간).
